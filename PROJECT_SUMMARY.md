# Project Summary - InnovatEPAM Portal

## Overview
InnovatEPAM Portal is a full-stack web application for submitting, evaluating, and tracking innovative ideas within an organization. Users can submit ideas, attach files, and participate in a structured evaluation workflow.

- [x] User Authentication - Complete

- [x] Evaluation Workflow - Complete

## Technical Stack

- **Framework**: React (frontend), Express (backend)
- **Database**: PostgreSQL (via Prisma ORM)
